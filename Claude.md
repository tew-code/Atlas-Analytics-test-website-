# ATLAS ANALYTICS v2 — Project Context

## What This Is
A marketing website for ATLAS ANALYTICS, a B2B sales pipeline analytics company.
Pure HTML/CSS/JS site (no frameworks, no build tools).

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- CSS Custom Properties for theming
- IntersectionObserver for scroll animations
- Decap CMS (admin/config.yml) for content management

## Design System
- **Colors:** Warm beige (#f0eee9) background, near-black (#1c1c1c) text, accent (#c9a227)
- **Fonts:** Space Grotesk (headings), Inter (body) — loaded via Google Fonts
- **Style:** Editorial minimal, lots of whitespace, clean typography

## File Structure
- 12 HTML pages (index, solutions, how-it-works, pricing, case-studies, about, blog, contact, privacy, terms, gdpr, 404)
- `css/style.css` — all styles
- `js/main.js` — all interactions
- `images/logo.svg` — SVG logo
- `admin/` — Decap CMS config

## Rules
- Always invoke the frontend design skill before writing any frontend code — every session, no exceptions
- Always test changes on localhost first
- Do NOT push to GitHub unless I explicitly tell you to
- Take screenshots while building to self-review your work
- Keep the editorial minimal design aesthetic — no cluttered UI
